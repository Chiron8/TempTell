# CPU Temp Display
A shell script to display CPU temperature in a customizable way.

## Installation
Clone this repository and run `makepkg -si` to install.

### Detailed Install:
First: `git clone https://github.com/Chiron8/TempTell`
Then: `cd TempTell`
Finally: `makepkg -si`

## Usage

```bash
cpu-temp [-c COLOR] [-f FORMAT]
